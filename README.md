# Smart Warehouse Management System

A C++ console-based Smart Warehouse Management System using Data Structures and Algorithms. The system manages product inventory, pending deliveries, delivery history, and shortest route calculation from Islamabad HQ.

## Features

* Product inventory management using Binary Search Tree
* Pending delivery management using Queue
* Completed delivery history using Stack
* City delivery network using Graph
* Shortest route calculation using Dijkstra’s Algorithm
* Menu-driven console interface

## Data Structures Used

* BST: Stores products by Product ID
* Queue: Stores pending deliveries
* Stack: Stores completed delivery history
* Graph: Represents cities and road distances
* Priority Queue: Used in Dijkstra’s Algorithm

## Technologies Used

* C++
* Data Structures and Algorithms
* Graph Algorithms
